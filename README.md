A comprehensive React-based design system tailored for enterprise data exchange applications. The Exchange Design System provides the foundational components, design tokens, and guidelines required to build consistent, accessible, and high-performance user interfaces.

## 🚀 Quick Start

Ensure you have Node.js (v18+) installed.

```bash
# Clone the repository
git clone https://github.com/rwyatt2/exchange-design-system.git

# Navigate into the directory
cd exchange-design-system

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Visit `http://localhost:5173` to view the interactive documentation site.
